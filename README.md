# Snapcode
Code Snippets Manager

## Demo
[Demo](https://snapcode.now.sh/)

## Features
- Manage your snippet
- Backup snippets to the cloud
- Shareable snippet

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
